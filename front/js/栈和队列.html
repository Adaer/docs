                          # 栈和队列

                          # 栈

                          ECMAScript为数组提供的push pop方法实现类似 “栈” 的行为 数据访问规则：LIFO（last-in-frist-out） 后进先出

                          TIP

                          数据在内存中的存放位置：变量栈 引用堆

                          # 实现栈的行为

                          let a = []
let count = a.push('data1', 'data2') //推入2项
count = a.push('data3') //推入3项

let item = a.pop() //从栈的后面取出
// item -> data3
a.length //2

                          # 队列

                          队列数据结构访问规则：FIFO（first-in-first-out）先进先出

                          # 实现队列的行为

                          let a = []
let count = a.push('data1', 'data2') //推入2项
count = a.push('data3') //推入3项

let item = a.shift() //从队列的前面取出
// item -> data1
a.length //2
